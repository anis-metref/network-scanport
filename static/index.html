À propos de Network Scanner

Network Scanner est une application moderne de scanning réseau développée avec Python et une interface web intuitive.

Fonctionnalités

  • Scan rapide et complet des réseaux
  • Détection des ports ouverts
  • Interface moderne inspirée de GNOME
  • Résultats en temps réel
  • Export des résultats
  • Historique et favoris avec base SQLite3

Technologies

Backend: Python, FastAPI, WebSockets, SQLite3
Frontend: HTML5, CSS3, JavaScript moderne
